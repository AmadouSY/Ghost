Install
-------
.. code-block::

    javac Ghost.java

Play
----

.. code-block::

    java Ghost

Then use the mouse.

Use ``-h`` to check the list of available options, and ``--rules`` for the
game rules. Press ``<space>`` at any time during the game to pause the game
(since 0.2.5).

Tests
-----

Clone this repo, then: ::

    [sudo] make deps
    make check
